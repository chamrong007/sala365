Introduction to Networks

  • 1. Network Fundamentals:
    - OSI and TCP/IP models
    - IPv4/IPv6 addressing & subnetting
    - Cabling and physical interfaces
    - Wireless principles
  • 2. Network Access:
    - VLANs and trunking
    - Spanning Tree Protocol (STP)
    - EtherChannel
    - Wireless LAN access
  • 3. IP Connectivity:
    - Routing concepts
    - Static and dynamic routing (OSPFv2)
    - First Hop Redundancy Protocols (FHRP)
  • 4. IP Services:
    - NAT, DHCP, NTP, DNS
    - SNMP, Syslog
    - QoS concepts
  • 5. Security Fundamentals:
    - Device access control
    - VPN concepts
    - Layer 2 security features
  • 6. Automation & Programmability:
    - Controller-based networking
    - REST APIs and JSON
    - Cisco DNA Center basics

Switching, Routing, and Wireless Essentials

  • 1. Spanning Tree Protocol:
    - STP Concepts & Operations
    - Varieties of STP
    - STP Configuration & Troubleshooting
  • 2. EtherChannel:
    - Link Aggregation Concepts
    - LACP and PAgP
    - Configuration and Verification
  • 3. Dynamic Host Configuration Protocol (DHCPv4):
    - DHCPv4 Concepts & Configuration
    - DHCPv4 Troubleshooting
  • 4. Wireless LAN Concepts:
    - Introduction to Wireless
    - WLAN Components
    - WLAN Operation & Security
  • 5. Routing Concepts & OSPF:
    - Path Determination
    - Dynamic Routing Protocols
    - Single-Area OSPFv2
  • 6. First Hop Redundancy:
    - Concepts of Redundant Default Gateways
    - HSRP, VRRP, GLBP

Enterprise Networking, Security, and Automation

  • 1. Advanced OSPF:
    - Multi-area OSPF
    - OSPF Tuning and Troubleshooting
  • 2. Access Control Lists (ACLs):
    - Standard and Extended ACLs
    - ACLs for IPv6
    - Troubleshooting ACLs
  • 3. Network Address Translation (NAT):
    - Static, Dynamic, and PAT
    - NAT Configuration
  • 4. WAN Technologies:
    - Introduction to WANs
    - VPNs and IPsec
    - QoS Concepts
  • 5. Network Management & Automation:
    - Syslog, SNMP, NetFlow
    - Introduction to SDN & APIs
    - Configuration Management Tools
